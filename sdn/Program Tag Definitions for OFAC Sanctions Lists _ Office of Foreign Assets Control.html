Program Tag Definitions for OFAC Sanctions Lists

The program codes or "tags" following each sanctions list entry indicate the sanctions program pursuant to which the person has been blocked, designated, or identified. Each sanctions list entry may have multiple program tags and may also appear on more than one list.
